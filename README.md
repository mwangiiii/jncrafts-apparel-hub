# Jncrafts Apparel Hub

Welcome to the Jncrafts Apparel Hub codebase! This project powers a modern, feature-rich apparel e-commerce platform built with React, TypeScript, Vite, and Tailwind CSS. Our goal is to deliver a fast, delightful shopping experience for users and a robust management interface for admins.

## Features
- Beautiful, responsive UI for shoppers and admins
- Product catalog with categories, featured products, and new arrivals
- Cart and checkout flows, including Mpesa payment integration
- User authentication and profile management
- Real-time chat and messaging widgets
- Supabase integration for backend services
- Performance optimizations for images and product listings
- Admin dashboard for product management

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm or bun

### Installation
1. Clone the repository:
   ```sh
git clone https://github.com/mwangiiii/jncrafts-apparel-hub.git
   ```
2. Install dependencies:
   ```sh
npm install
   # or
bun install
   ```
3. Start the development server:
   ``` npm run dev

npm run dev
   # or
bun run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
- `src/` — Main source code
  - `components/` — UI components
  - `contexts/` — React context providers
  - `hooks/` — Custom React hooks
  - `pages/` — Page components (routing)
  - `integrations/` — External service integrations
  - `lib/` — Utility libraries
  - `assets/` — Static assets
- `public/` — Static files served directly
- `supabase/` — Supabase config and migrations

## Scripts
- `dev` — Start development server
- `build` — Build for production
- `lint` — Run ESLint
- `test` — Run tests (if available)
